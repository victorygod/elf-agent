# 生存型 Agent 设计文档

本文记录一种以"在环境中独立生存"为目标的 agent 架构讨论结论。起点是对现有 elf 双层架构(Intuitive / Reasoning)的重新定位,终点是一套以生物分层神经控制为蓝本的可行方案草图。

---

## 一、对现有双层架构的重新定位

### 1. 现状

elf 当前的双层架构(`shared/agent/default_agent.js`):

- **Intuitive 层** = `receive()`,目前是 pass-through,收到消息直接转发给 reasoning。无 loop、无工具、无 memory。
- **Reasoning 层** = `reasoning()` 的 agent loop(`LLM → 工具调用 → LLM → … → 回复`),承载了全部 prompt 构建、记忆读写、工具调用。

两层共用同一份 prompt / tools / memory。问题在于:**真正的 loop 只活在 reasoning 里,Intuitive 还没进入循环**。

### 2. 目标定位

我们要的不是"对话型 LLM agent",而是**能在环境中独立生存的 agent**。重新定位如下:

- **Reasoning(LLM 推理)不是 agent 的全部,只是它的一种能力。** Agent 的核心是 intuitive——结合环境输入与自身状态作出反应。LLM 出现前,agent 的完整状态就是"在环境中,根据自身状态、环境状态及二者变化与关系,作出行动";LLM 出现后,agent 能力得到扩充:当情形复杂到传统决策行为树无法解决时,才依赖 LLM 的 reasoning。
- **Intuitive 自带完整身体控制**,因此它自己就能跑完一整轮 agent 循环(感知→反应→行动)。
- **Reasoning 是异步的、可降格为"感官"的。** Intuitive 判断需要 reasoning 时,**立刻响应、不等结果**;reasoning 返回后,在下一轮循环里作为"自身环境变化的一种"被 intuitive 重新感知。这把 reasoning 从一条特殊通道,降格成了一种感知来源。
- **统一的心智模型**:外部世界、自身状态、工具副作用、reasoning 结论、他人消息——全是 observation;唯一出去的通道是 action。tool_call / tool_result 的消息缠绕被整个砍掉,action 出、observation 进,两条干净的单向通道。

### 3. 极简生存 loop

```
while True:
    obs = perceive(env, self)        # 外部状态 + 自身状态(含知识缓冲 + 在途 reasoning 回执)
    out = intuitive(obs)
    act = parse_action(out)          # act | escalate(带状态快照) | null
    execute(act)                     # 改世界 or 发 reasoning 请求(不阻塞)
    # reasoning 回执异步写回 self-state,下一轮被 perceive 读到
```

这是一个比 ReAct 更诚实的生存模型:agent 始终 grounded 在世界里,而非漂浮在一条对话消息流上。

## 二、分层 attention 闸门(多级感知 / 反射漏斗)

"每轮必调 LLM"会让 fast 路径不 fast。真正的生存反射是亚百毫秒级的,扛不住任何 LLM 往返。因此感知入口是一条**逐级提级的漏斗**,每一级便宜一档,只有上一层判定为值得关注才进下一层,大部分在前两层就被短路掉:

```
L0  状态差分 / 显著度门        纯确定性,无模型
    → 无变化 / 低显著度 → null action,continue   (绝大多数 tick 死在这)

L1  反射级判断                 极便宜的小模型 / 规则,单次,无推理
    → 能直接 act 的 → 执行反射动作
    → 把握不准 → 提级 L2

L2  intuitive 局势分析          更聪明但仍单次、低延迟的小模型
    → 给出 reflex act 或 escalate(带状态快照)

L3  reasoning                  大模型,多步,异步
    → 回执写回 self-state,L1/L2 下轮感知
```

- **L0 必须确定性,且必须输出"变化差分"而非 bool**:计算状态 diff,显著 diff 才往 L1 喂。否则要么每轮全量喂 LLM(成本 + 注意力稀释),要么漏掉"零变化但持续"的慢漂移。
- **提级要带置信度 / 成本预算,而非裸传**:提级信号应是一等输出(如"我在哪类情况上不确定"),既给上层指明注意力,又作为"某类局势总过不了下层→该下沉成规则"的标定反馈,构成自我收敛回路。
- **L0/L1 必须能直接产 action**:否则 fast 路径是死的,全程坍到 L2/L3。L1 的反射动作表是命门——高频、低风险、可快速执行 / 可回滚的 action 原语。

### 隐藏好处:层级天然抗 reasoning 风暴

reasoning 回执不是直接插进 L2,而是先进 L0 被当作"一条自身状态变化"做显著度判断。若此刻局势已变、它已无关 → L0 判低显著 → 直接丢弃,**永远不到 L2 干扰当下判断**。这把"过期 reasoning"问题拆解掉了:不靠上层显式判断"还适不适用",而靠 L0 显著度门把过期、已被环境覆盖的建议自然过滤——比显式元认知便宜,也更"生物"。

## 三、与生物进化的对照(关键修正)

对照古生物→现代生物的神经控制进化,验证并修正了前述设计。结论:**进化的时序梯度(越低越快越老)与 L0~L3 映射良好,但生物的控制结构不是"提级漏斗"**。

### 1. 进化时序 vs L0~L3(对得上)

| 进化阶梯 | 控制 | 对应层 |
|---|---|---|
| 细菌(化学趋向) | 分子梯度→鞭毛翻转,无神经 | L0 |
| 刺胞(水母/水螅) | 弥散神经网,触→缩,局部反射弧 | L0 |
| 环节 / 节肢动物 | CPG(内源节律振荡,走/呼吸/咀嚼自发)、侧抑制 | L0/L1 |
| 脊椎动物脊髓 / 脑干 | 脊髓反射(膝跳/缩手不上脑)、存活反射 | L0/L1 |
| 基底神经节 | 动作选择,winner-take-all 解除竞争性运动程序抑制 | L1/L2 选择器 |
| 边缘系统(杏仁核/下丘脑) | 杏仁核"低路"绕皮层做快速恐惧,valence 标注,但可学 | L1/L2 |
| 前额叶皮层 | 工作记忆、前瞻规划 | L3 |

梯度规律:越底层越快、越省能、越老、越不可逆要先执行;皮层贵且慢,生物经济学要求把绝大部分生存行为下沉到反射/习惯,皮层只在罕发且高代价处接管。这支撑了"大部分 tick 不该上 LLM"的设计直觉。

### 2. 三处对不上(修正点)

**① 并发抑制(subsumption),不是提级漏斗——最重要。**
生物不"报上级":遇火缩手在脊髓就执行了,皮层事后才知晓,只能事后 veto/调制。进化从不删低层,而是**加高层去抑制/覆盖**它。CS 的正式化是 Brooks 的 subsumption——所有层**同时运行**,高层可 suppress 低层,但低层始终活跃并抢先。
→ 直接后果:实现不该是瀑布式 if-escalate,而应是**并行候选 + 优先级抑制**——L0/L1 先动,只有它们抑制不住或主动让出时,reasoning 才补位。这是 loop 主干的选择。

**② "显著度"之下还缺一层:内驱 / 价值(homeostatic drive)。**
不存在中立显著度——什么显著取决于内驱。饿→食物显著;受威胁→威胁显著。valence 来自古老神经调质(多巴胺/5-羟色胺)和下丘脑内稳态 set-point。→ L0 之下应埋一层显式 homeostatic drive(饥饿/恐惧/好奇等 set-point),正是它给显著度提供判据。
- 这正好对应刻意区分的"自然界生存 vs 内部需求生存"两类:生物上确是两套古老子系统(感觉运动-空间系 vs 下丘脑内稳态系),二者汇聚到基底神经节同一动作选择器。
- **高层可塑低层注意力**:人会"集中注意力专注倾听"——这是高层(皮层意图)塑造低层感知敏感度的例子。注意力并非纯底层现象,内驱与高层意图都能自上而下调制显著度门。设计中应允许显著度判据被内驱状态与上层意图共同动态设定。

**③ 低层可塑,且 L3 会向 L0/L1 固化——最反直觉也最有价值。**
习惯/恐惧条件反射是真实脑机制(背侧纹状体习惯环路由多巴胺 RPE 强化,把原本缓慢审慎的练习固化成快速自动反应)。即生物的"反射"大量是**学来**的。→ 若 L0/L1 是纯死规则,既不符合生物也自缚手脚。应给一条 **L3→L0/L1 的下行学习/固化通道**:reasoning 解出的成功策略,经强化逐步下沉成 reflex 规则。结果是 agent **越用越快**(而非仅越用越聪明),因为 learning 沉淀进 reflex 层,而非堆进对话上下文。这条双向(上行 escalation / 下行 consolidation)是与生物一致、且构成护城河之处。

### 3. 修正后的结构

把 L0~L3 从"提级漏斗"改为 **"并行抑制 + 内驱驱动 + 下行固化"**:

- 各层并发存在,逐时刻都跑,靠优先级 / veto,而非串行上报;
- L0 之下加一层显式 homeostatic drive——它既给显著度提供判据,也驱动"即使无外界刺激也要 act"的内生 tick 值(对应 `while True` 的动力:tick 来自内驱未满足,而非外部信号);
- 提供一条 L3→reflex 的固化通道。

## 四、开放决策(待定)

1. **控制结构:漏斗 vs subsumption。** 漏斗简单、调试友好;subsumption 更生物、下层永远兜底更鲁棒,但需设计层间抑制/仲裁机制。倾向 subsumption 的退化起步版——"反射优先,reasoning 仅作为可被低层 suppress 的一类并行候选"。
2. **L3→reflex 固化通道。** 决定了是"会快点的 LLM agent"还是"会成长出本能的生存体"。生物给的答案是后者才有戏。
3. **L1 是否仍以 LLM(小模型)实现直觉。** 走"小一点的同款推理"(轻,但反射不真反射)还是走"分层确定性反射 + 规则表"(更像生物生存体,但要写/标定反射表)。倾向后者——否则与外面 agent 无本质区别。

## 五、下一步:可行性方案的前置——环境与感受器 / 效应器定义

在讨论注意力分层可行方案前,需先设定:

1. **agent 的生活环境**(agent 生活在什么世界里?物理 / 信息 / 混合?有什么威胁与资源?时间尺度?)
2. **对外、对内的感受器**(agent 能感知什么?外部状态与内部状态各有哪些观测量?)
3. **行动能力 / 效应器**(agent 能做什么?哪些是改变外部世界的 action,哪些是改变自身状态的 action?)

只有这三者初步定义,注意力分层的可行方案才有意义——脱离具体环境的分层设计是空中楼阁。
